VideoCrop

Made by Dennis Kats
View the source on GitHub

VideoCrop is an offline, browser-based video cropping tool built on top of ffmpeg.wasm. In fact, VideoCrop will never upload or send anything to a server, and works even if you disable your Wi-Fi. Start by clicking on the file input button or by dragging and dropping a video file onto the page.

Keyboard Shortcuts (while the "area selector" is visible and focused...)
  • Use ↑,→,↓,← (the arrow keys) to move the "area selector" by 1 pixel.
  • Hold ⌘/⊞ (the meta key) to change the modifier value to 10 pixels.
  • Hold ⇧ (the shift key) and ⌘/⊞ (the meta key) together to change the modifier value to 25 pixels.
  • Hold ⌥/Alt (the alt key) and Ctrl (the control key) with any of the above combinations to adjust the size of the "area selector".
